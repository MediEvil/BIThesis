## 项目内容

SHUThesis 为各位在上海大学就读的本科同学提供了上海大学本科生毕业设计（论文）模板。借助于 SHUThesis 的 LaTeX 模板，你可以在保证论文格式整齐、完美、符合要求的前提下，专注于学术研究、项目实现，从而顺利完成你的学术项目。
## 如何使用 SHUThesis
